03 / Context7

Что такое Context7?

Context7 — MCP-сервер, который подаёт Claude актуальную документацию библиотек прямо в промт. Без него Claude отвечает по данным обучения — которые могут быть устаревшими.

❌ Без Context7
✅ С Context7
Как это работает
1
Установи MCP
npx -y @upstash/context7-mcp
Добавь в .claude/settings.json
2
Добавь в промт
"... use context7"
Context7 автоматически загрузит актуальные доки
3
Получи точный ответ
✓ Реальный API
✓ Текущая версия
✓ Без галлюцинаций
Claude отвечает по свежей документации
Конфигурация MCP (settings.json)
{ "mcpServers": { "context7": { "command": "npx", "args": ["-y", "@upstash/context7-mcp"] } } }
